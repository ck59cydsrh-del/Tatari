MENU — 音源MENU — SOUND SOURCE
DEMO=内蔵のテスト用ビート。FILE=スマホ内の曲。MIC=マイクをそのまま通す。どれもタップで開始、もう一度タップで停止。何も選ばなければ無音。破壊の対象は「いま鳴っているもの」。DEMO = built-in test beat. FILE = a song from your phone. MIC = live microphone. Tap to start, tap again to stop. Nothing selected = silence. Whatever is playing is what gets destroyed.
CHEW パッドTHE CHEW PAD
押さえている間だけ音が壊れ、離すと即ドライに戻る。指の位置が X=JINX(事故率) / Y=BITE(刻み幅)。左下が安全、右上が最凶。HOLDをオンにするとタップでラッチ(押しっぱなし状態を固定)。PCならスペースキーでも作動。The sound breaks only while you hold it; release and it snaps back. Finger position = X = JINX (accident rate) / Y = BITE (slice length). Bottom-left is safe, top-right is maximum damage. Turn on HOLD to latch it with a tap. Spacebar works on desktop.
数字の帯(上段)THE NUMBER STRIP
5つの数字はすべて上下ドラッグで調整、ダブルタップで初期値(BITE以外)。BITE欄はタップでロック(白黒反転)— ロック中はパッドやピンチで刻み幅が動かなくなり、パッドがX(JINX)専用になる。もう一度タップで解除。All five numbers adjust by dragging up/down; double-tap resets (except BITE). Tap the BITE cell to lock it (inverted display) — while locked, the pad and pinch can no longer change the slice length, turning the pad into a JINX-only fader. Tap again to unlock.
JINX — 事故率JINX — ACCIDENT RATE
00–30 同じスライスを同じピッチで反復するだけのクリーンな連打。30–65 逆再生・半速/倍速・位置ジャンプが混ざり始める。65–99 テープストップ、ラチェット(刻みが勝手に細かくなる加速ロール)、一瞬の無音、完全ランダム化まで全解禁。00–30 clean beat-repeat: same slice, same pitch. 30–65 reverse, half/double speed and position jumps start creeping in. 65–99 everything unlocked: tape stops, ratchets (self-accelerating rolls), dropouts, full scramble.
BITE — 刻み幅BITE — SLICE LENGTH
10ms〜800ms。低=ブザー的な音程感 / 中=16分ロール / 高=フレーズごとのループ。SYNCオン中は音価(1/16など)にスナップ。10ms–800ms. Low = buzzy pitch, mid = 16th-note rolls, high = phrase loops. With SYNC on it snaps to note values (1/16 etc).
ROT — 腐敗ROT — REGENERATION
00–40 反復ごとに減衰、一発フィル向き。40–75 減衰なしの無限ロール。75–99 フィードバックが臨界を超え、出力が録音バッファに再侵入して飽和しながら自壊。スコープ右上のGENが反復の世代数。00–40 each repeat fades — one-shot fills. 40–75 infinite roll. 75–99 feedback goes past unity: the output re-enters the buffer, saturates, and eats itself. GEN (top-right of the scope) counts the generations.
BURN — 破壊BURN — DESTRUCTION
00–40 ドライブのみ。40–75 +ダウンサンプルとビット削減。75–99 +ウェーブフォールドとノイズ。全開で4bitの粉末。00–40 drive only. 40–75 + downsampling and bit reduction. 75–99 + wavefolding and gated noise. Full = 4-bit powder.
IN — 入力ゲインIN — INPUT GAIN
入力の音量。33=等倍、最大3倍。スコープ下端の黒いバーが入力メーターで、MICで喋ってバーが振れれば受信できている。Input volume. 33 = unity, up to 3×. The black bar at the bottom of the scope is the input meter — if it moves while you speak into MIC, the mic is working.
SYNC / BPM / TAP — テンポ同期SYNC / BPM / TAP — TEMPO SYNC
SYNCをオンにすると刻み幅がBPM基準の音価(1/32〜1BAR)にスナップし、ロールが曲のグリッドに揃う。TAPを曲に合わせて叩くと自動でBPM検出+SYNCオン。BPMの数字はタップで直接入力、上下ドラッグで微調整。欲しい音価が出たらBITE欄タップでロック、が基本コンボ。With SYNC on, slice lengths snap to note values (1/32–1 bar) of the BPM, so rolls land on the grid. Tap TAP along with your track to auto-detect BPM and enable SYNC. Tap the BPM number to type a value, or drag it. The go-to combo: find the note value you want, then tap BITE to lock it.
SMPL — 録ってループ化SMPL — SAMPLE & LOOP
● SMPLで今鳴っている入力(MICの声など)の録音開始、もう一度押すと停止して自動でループ再生に切り替わる(最長30秒)。録った声をループさせてじっくり破壊できる。CLEARで破棄。● SMPL starts recording whatever input is playing (e.g. your voice on MIC); press again to stop and it automatically becomes a playing loop (max 30s). Now destroy your own voice at leisure. CLEAR discards it.
REC — 演奏をWAVで保存REC — RECORD TO WAV
● RECで出力の録音開始、停止するとSAVE .WAVが出る。タップして共有シートから「ファイルに保存」。画面収録には音が乗らないので、音を残すならこれを使う。最長5分。● REC records the output; stop and SAVE .WAV appears — tap it and choose "Save to Files". Screen recording does NOT capture this audio, so use REC instead. Max 5 minutes.
サイドボタンSIDE BUTTONS
HOLD=パッドをラッチ式に。SHAKE=振り検出のオン/オフ(初回タッチ時に許可を求めて自動オン)。TILT=傾き操作のオン/オフ。HOLD = latch the pad. SHAKE = shake detection on/off (auto-enabled after permission on first touch). TILT = tilt control on/off.
こまったらTROUBLESHOOTING
音が出ない → MENUで音源を選んでいるか確認。マイクが効かない → スコープ下のバーが振れるか確認、振れなければSafariのサイト設定でマイクを許可。振っても壊れない → SHAKEボタンを一度オフ→オンで再許可。意図しない無音は約0.2秒で自動復帰する設計。No sound → make sure a source is selected in MENU. Mic not working → check the meter bar under the scope; if it doesn't move, allow microphone access in Safari's site settings. Shake not working → toggle SHAKE off and on to re-request permission. Unintended silence auto-recovers within ~0.2s by design.
